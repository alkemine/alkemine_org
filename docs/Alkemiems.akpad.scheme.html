Module contents¶

Scheme¶

The scheme package implements and defines the underlying workflow model.

The Scheme class represents the workflow and is composed of a set of SchemeNode connected with SchemeLink, defining an directed acyclic graph (DAG). Additionally instances of SchemeArrowAnnotation or SchemeTextAnnotation can be inserted into the scheme.
