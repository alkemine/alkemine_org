Alkemiems.lammps.core.get_default_cmd_set_warn(cmd, dir)[source]¶

输入一个要获取的命令，比如lmp_cmd，出入一个要索引的字典a={} 如果a里面有cmd对应的参数，返回改参数值， 如果a中没有cmd参数，会提示警告，并通过全局变量查找结果并返回

eg:

dir = {‘lmp_cmd’: 123} get_default_cmd_set_warn(‘lmp_cmd’, dir) 返回123

dir = {} 全局变量LMP_CMD = ‘ggggg’ get_default_cmd_set_warn(‘lmp_cmd’, dir) 返回’ggggg’

Args:
cmd: 全部小写字符串 dir: 要索引的字典

Returns: 变量值
