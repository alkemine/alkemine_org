改变高通量任务队列¶

通过 Work Priority 控件，可以调整工作流计算顺序。首先，连接 Manager 控件到 Work Priority ，然后进入界面，输入Job Id，输入Job priority（0-100，100为优先级最高），按 Submit 即可调整工作流顺序。
