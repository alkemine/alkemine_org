编译环境¶

Windows安装环境¶

  • 在编译之前必须要有numpy，否则C语言相关脚本无法编译。

  • 建议使用anaconda安装python3.7.0的环境

  • Visual Studio 14++ (Windows)

Linux安装环境¶

  • 在编译之前必须要有numpy，否则C语言相关脚本无法编译。

  • gcc (GCC) 4.4.7 20120313 (Red Hat 4.4.7-4)

  • 建议使用anaconda安装python3.7.0的环境

  • 可视化界面需要：glibc/2.16

Note

接下来具体安装步骤均以Windows10为主说明。
