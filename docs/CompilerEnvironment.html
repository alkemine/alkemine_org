编译环境¶

Windows安装环境¶

  1. python（建议使用Anaconda安装python3.7.0的环境）

  2. numpy（安装ALKEMIE之前必须预先安装numpy，否则c语言相关脚本无法编译）

  3. Visual Studio 14++ (C和C++编译依赖)

Linux安装环境¶

  1. python（建议使用Anaconda安装python3.7.0的环境）

  2. numpy（安装ALKEMIE之前必须预先安装numpy，否则c语言相关脚本无法编译）

  3. gcc (GCC) 4.4.7 20120313 (Red Hat 4.4.7-4)

  4. 可视化界面需要：glibc/2.16

Note

接下来具体安装步骤均以Windows10为主说明。
