蜂窝状格点上原子的扩散¶

该算例根据第一性原理计算的空位扩散的势垒，模拟介观尺度的模特卡洛模拟，并获得原子的扩散系数。计算完成第一性原理NEB计算之后，链接到 Kmc Run 执行计算，最后通过 Kmc Plot 画图，完整流程图如下：

  • 第一步，手动连接控件，或点击 File ， Open and Freeze ，点击打开对应样例模板文件并点击开始按钮，如图：

  • 第二步，选择结构，点击 MultiPOSCAR ，选择空位扩散的两端点结构；点击 POSCAR 选择最原始的结构（Parent Structure）

  • 第三步，点击左下角 开始 按钮，运行neb计算工作流；

  • 第四步，在Jobs模块内点击 Run jobs ，提交任务，点击 Jobs 查看记录任务号；

  • 第五步，点击 Get_delta_E ，输入jobs_id，获得能量势垒，如图：

  • 第六步，点击 Run honeycomb lattice Kmc WF， 配置参数，如图：

  • 第八步，在Jobs（1）提交任务，等待计算结束；

  • 第九步，点击 Get Kmc Data ，输入jobs_id，即可传输数据到Kmc Plot，如图：

  • 第十步，点击 Kmc Plot ，绘制图像，如图：
