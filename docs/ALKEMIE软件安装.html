ALKEMIE软件安装¶

ALKEMIE软件包95%语言通过python语言开发完成，另外还包含少量C语言和C++语言代码，目前服务器及数据库已经部署在北航学院路校区超算集群、北航沙河校区超算集群及广州国家超算中心共三个超算中心；客户端版本可以在Windos7, Windows10, MacOS high sierra 10.13.6 及 Ubuntu18.04.2等系统完美运行，软件可移植性较好。如果使用者熟悉python语言，可以创建一个全新的python环境，通过pip install 轻松安装，由于有部分C和C++代码，因此需要VS14++或者GCC4.4.7+的编译器编译部分C及C++代码。接下来的部分详细介绍ALKEMIE的安装步骤。
