Install Visual Studio Installer¶

Download the Visual Studio 2019 installation package at https://visualstudio.microsoft.com/zh-hans. Double-click the mouse to start the installation program, choose to use C++ desktop development, and install it according to the default path. The installation process takes about half an hour. After the installation is complete, follow the prompts to restart the computer.
