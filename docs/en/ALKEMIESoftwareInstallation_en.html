Installation¶

95% of the ALKEMIE software package is developed in python language, and it also contains a small amount of C and C++ language codes. Currently, the server and database have been deployed in the Beihang College Road Campus Supercomputing Cluster, Beihang Shahe Campus Supercomputing Cluster and Guangzhou National Supercomputing Center ; The client version can run perfectly on systems such as Windos7, Windows10, MacOS high sierra 10.13.6 and Ubuntu 18.04.2. The software portability is good. If users are familiar with the python language, they can create a new python environment and install it easily through pip install. Because there are some C and C++ codes, they need VS14++ or GCC4.4.7+ compiler to compile some C and C++ codes. The following part details the installation steps of ALKEMIE.
