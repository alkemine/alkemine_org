2. **Introduction to ALKEMIE**

ALKEMIE（Artificial Learning and Knowledge Enhanced Materials Informatics Engineering）It is a high-throughput automatic process computing software with multi-scale integration function. Led by Professor Sun Zhimei of Beijing University of Aeronautics and Astronautics, it is a set of completely independent high-throughput automatic process computing software based on Python open source framework. The software includes two versions: stand-alone version (full function) and web page version (database function). Users can download the stand-alone full version after registering on the web page.
